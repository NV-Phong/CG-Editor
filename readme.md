<h1 align="center">CG-EDITOR</h1>

### INSTALL DEPENDENCIES

```sh
pnpm install
```

### RUN PROJECT

```sh
# development
pnpm dev

# production
pnpm build
# then run
pnpm --filter cg-editor start
```
