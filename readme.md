<h1 align="center">CG-EDITOR</h1>

### INTRODUCE

This Contribution Graph Editor, built with NextJS, provides an intuitive interface with auto-commit functionality.

### INSTALL DEPENDENCIES

```sh
pnpm install
```

### RUN PROJECT

```sh
# development
pnpm dev

# production
pnpm start
```
